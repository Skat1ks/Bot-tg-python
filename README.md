# 📚 Dictionary and Translator Bot

## 📖 Description

**Dictionary and Translator Bot** is a Telegram chatbot created with Python.  
The bot allows users to add, search, delete, and manage words with their meanings or translations.

The project was developed as a final project for the дисциплина **“Programming in Python”**.

The bot supports:
- adding words and translations;
- searching words;
- deleting words;
- viewing all saved words;
- saving user history;
- displaying statistics;
- handling incorrect input and errors.

---

## 🎯 Project Goal

Create a Telegram chatbot with a user-friendly interface and implemented interaction logic using Python.

---

## ⚙️ Technologies Used

- Python 3
- Telegram Bot API
- python-telegram-bot
- SQLite3
- JSON
- Datetime module
